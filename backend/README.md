# 💰 AI Finance Tracker

An AI-powered full-stack finance management application developed using React.js, Flask, and Machine Learning to track, manage, and predict user expenses.

## 🚀 Features
- Add and manage expenses
- Expense analytics dashboard
- AI-based spending prediction
- Savings tracker
- Budget alerts
- REST API integration
- Automated API testing
- Docker containerization
- Jenkins CI/CD pipeline

## 🛠 Technologies Used
Frontend: React.js, Chart.js, CSS  
Backend: Flask, Python, Machine Learning  
DevOps: Docker, Jenkins, GitHub
